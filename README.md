# Proka Bootloader Template
This repository is the template for Proka Bootloader.

For more information, please visit the [Proka Bootloader](https://github.com/proka-bootloader/proka-bootloader) repository.

## Usage
If you want to build this, just run this command:

```bash
make all
```

and the file will be put in `proka-kernel` in the current directory.
